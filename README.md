# Fraud Detection in Financial Transactions

## Overview
This project aims to detect fraudulent financial transactions using machine learning techniques. The model classifies transactions as legitimate or fraudulent based on transaction data.

## Dataset
The dataset contains transaction-related features such as amount, time, and other anonymized variables. The target variable indicates whether a transaction is fraudulent (1) or legitimate (0).

## Methodology
- Data preprocessing and handling class imbalance
- Exploratory Data Analysis (EDA)
- Model training using classification algorithms
- Performance evaluation using accuracy, precision, recall, and F1-score

## Results
- Model evaluated on unseen test data
- Performance measured using classification metrics
- Demonstrates effective detection of fraudulent transactions

Detailed metrics are available in `results.txt`.
